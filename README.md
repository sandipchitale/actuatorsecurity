# Description

Apply different i.e. permitAll to Actuator endpoints. Integration with Zipkin.

## How to use with ZipKin

- Start Docker
- Run the following command to start Zipkin container 

```
docker-compose up
```

- See traces in Zipkin UI by using the following URL

```
http://localhost:9411/zipkin/
```
